# Movie Tracker App (React)

A React application that allows users to search for movies, view detailed information, and manage a personal watched list using data from the OMDb API.

---

## Features

- Search movies using the OMDb API
- View detailed movie information (plot, cast, rating, etc.)
- Add movies to a personal watched list
- Rate watched movies
- Remove movies from watched list
- Persist watched list using localStorage
- Conditional rendering for different UI states (loading, details view, list view)
- Keyboard shortcut support (custom hook for key handling)

---

## Tech Stack

- React
- JavaScript (ES6+)
- CSS
- OMDb API

---

## What I Learned

This project was initially built by following a React tutorial and used as a structured learning exercise. It helped me develop a deeper understanding of how React applications work in practice.

Key concepts learned:

- Component-based architecture in React
- State management using useState
- Side effects and data fetching using useEffect
- Props and data flow between components
- Conditional rendering for dynamic UI updates
- Managing persistent state using localStorage
- Data transformation and destructuring from API responses
